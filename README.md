## 📌 IPL Score Prediction using Machine Learning

This project is about **Predicting final score of an IPL cricket match** using Machine Learning techniques. The model analyzes match conditions such as batting team, bowling team, overs completed, runs scored, wickets fallen, and runs in the last five overs to estimate the final score.

In this project, different ML models such as **Linear Regression, Decision Tree Regression, Random Forest Regression, and AdaBoost Regression** were implemented and compared to determine the best-performing model.

After evaluation, **Linear Regression** provided the most accurate results and was used as the final prediction model.
]

## Features

* Data preprocessing and cleaning of IPL dataset
* Exploratory Data Analysis (EDA) to understand match patterns
* Implementation of multiple regression models
* Model evaluation and comparison
* Final score prediction based on live match parameters
* Function for predicting IPL scores using trained model


## Machine Learning Models Used

* Linear Regression
* Decision Tree Regression
* Random Forest Regression
* AdaBoost Regressor



##  Input Parameters for Prediction

The model predicts the final score using:

* Batting Team
* Bowling Team
* Overs Completed
* Current Runs
* Wickets Fallen
* Runs in Last 5 Overs
